# タグ操作

**Tag** パネルから Git タグを直接管理します。

## キーバインディング

| キー     | アクション                     |
| :------- | :----------------------------- |
| `< / >`  | コンポーネントパネルの切り替え |
| `d`      | タグを削除                     |
| `ctrl+p` | タグをプッシュ                 |
| `f`      | タグをフェッチ                 |
| `?`      | キーバインディングと手順       |

:::info
ローカルブランチ、タグ、リモートパネルは同じスペースを共有しています。`< / >` を使用してパネルを切り替えます。新しいタグを作成するには、コミットログパネルで `t` キーバインディングを使用します。
:::

## ワークフロー

### タグの削除

1.  **移動**: Tag コンポーネント（`1`を押す）に移動するか、Tag コンポーネントに切り替えます（`< / >`を押す）。
2.  **選択**: `↑/↓` を使用して削除するタグを選択します。
3.  **トリガー**: `d` を押してタグ削除メニューを開きます。
4.  **モードの選択**: 削除オプションを選択します：
    - ローカルタグを削除 (Delete local tag)
    - リモートタグを削除 (Delete remote tag)
5.  **確認**: `enter` を押して確定します。
6.  **閉じる**: `esc` を押して出力を閉じます。

---

### タグのプッシュ

1.  **移動**: Tag コンポーネント（`1`を押す）に移動するか、Tag コンポーネントに切り替えます（`< / >`を押す）。
2.  **選択**: `↑/↓` を使用してプッシュするタグを選択します。
3.  **トリガー**: `ctrl+p` を押してタグプッシュメニューを開きます。
4.  **モードの選択**: プッシュオプションを選択します：
    - 選択したタグをプッシュ (Push selected tag)
    - すべてのタグをプッシュ (Push all tags)
    - 選択したタグを強制プッシュ (Force push selected tag)
    - すべてのタグを強制プッシュ (Force push all tags)
5.  **確認**: `enter` を押して確定します。
6.  **閉じる**: `esc` を押して出力を閉じます。

---

### タグのフェッチ

1.  **移動**: Tag コンポーネント（`1`を押す）に移動するか、Tag コンポーネントに切り替えます（`< / >`を押す）。
2.  **トリガー**: `f` を押してタグフェッチメニューを開きます。
3.  **モードの選択**: フェッチオプションを選択します：
    - タグをフェッチ (Fetch tag(s))
    - タグをフェッチ・プルーン (Fetch tag(s) prune) - リモートに存在しないタグを削除
    - タグをフェッチ・上書き (Fetch tag(s) overwrite) - リモートタグと異なるローカルタグを上書き
    - タグをミラー (Fetch tag(s) mirror) - リモートからすべてのタグをミラーリング
4.  **確認**: `enter` を押して確定します。
5.  **閉じる**: `esc` を押して出力を閉じます。
