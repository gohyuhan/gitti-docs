# ブランチ管理

ローカルブランチを効率的に管理します。Gittiは、ガイド付きのフローでブランチの作成と切り替えを処理します。

## ナビゲーション
`1` を押して **ローカルブランチ** パネルにフォーカスします。

## キーバインド

| キー | アクション | コンテキスト |
| :--- | :--- | :--- |
| `enter` | ブランチを切り替え | 選択したブランチがチェックアウトされていない場合 |
| `n` | ブランチを作成 | ブランチパネルで常に利用可能 |

## ワークフロー

### ブランチの切り替え

1.  **選択**: `up`/`down` (`k`/`j`) を使用してチェックアウトしたいブランチに移動します。
2.  **トリガー**: `enter` を押します。
3.  **ロジックフロー**:
    - **すでにチェックアウトされている場合**: 何も起こりません。
    - **チェックアウトされていない場合**: 切り替え方法（例：標準チェックアウト）を尋ねるポップアップが表示されます。
4.  **確認**: 切り替えタイプを選択し、`enter` を押します。

### 新しいブランチの作成

1.  **トリガー**: `n` を押します。
2.  **ステップ 1: タイプの選択**: 作成モードを尋ねるポップアップが表示されます：
    - `New Branch`: 作成のみ。
    - `New Branch & Switch`: 作成してすぐにチェックアウト。
3.  **ステップ 2: 名前の入力**: タイプを選択した後、ブランチ名を入力する新しいポップアップが表示されます。
4.  **送信**: 名前を入力し、`enter` を押します。
