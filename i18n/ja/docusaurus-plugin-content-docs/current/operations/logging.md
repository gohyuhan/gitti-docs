# ログとデバッグ

Gitti は、問題のトラブルシューティングや Git 操作のリアルタイム監視に役立つ専用のログパネルを提供します。

## ログパネルへのアクセス

`/` を押して **ログパネル** にフォーカスします。

## キーバインド

| キー | アクション         |
| :--- | :----------------- |
| `e`  | ログのエクスポート |

## 機能

### ログのエクスポート

現在のセッションログをファイルにエクスポートして、外部での分析やバグ報告に使用できます。

1.  `/` でログパネルにフォーカスします。
2.  `e` を押します。
3.  ログファイルはシステムの **ダウンロード** フォルダに保存されます。

### 設定

コマンドラインフラグを使用して、ログの動作をカスタマイズできます：

- **最大ログ数 (Max Log Count)**: メモリに保持するログエントリの最大数を設定します。

  ```bash
  gitti --max-log-count 1000
  ```

- **表示ログ数 (Show X Log)**: パネルに表示する最近のログの数を設定します。
  ```bash
  gitti --show-x-log 50
  ```
