# スタッシュ

コミットされていない変更を後で使用するために保存します。

## ナビゲーション

`4` を押して **スタッシュ** パネルにフォーカスします。

## キーバインド

| キー        | アクション               | コンテキスト           |
| :---------- | :----------------------- | :--------------------- |
| `s`         | ファイルをスタッシュ     | 変更されたファイルから |
| `S`         | すべてスタッシュ         | 変更されたファイルから |
| `space`     | スタッシュを適用 (Apply) | スタッシュパネルから   |
| `backspace` | スタッシュをポップ (Pop) | スタッシュパネルから   |
| `d`         | スタッシュを削除 (Drop)  | スタッシュパネルから   |

## ワークフロー

### スタッシュの作成

1.  **フォーカス**: **変更されたファイル** (`2`) に移動します。
2.  **トリガー**:
    - `s` を押して **選択したファイルのみ** をスタッシュします。
    - `S` を押して **すべてのファイル** をスタッシュします。
3.  **ポップアップ**: "Stash Message"（スタッシュメッセージ）。スタッシュにわかりやすい名前を入力します。
4.  **送信**: `enter` を押します。

---

### スタッシュの管理

1.  **フォーカス**: **スタッシュ** (`3`) に移動します。
2.  **適用 (Apply)**: スタッシュを選択し、`space` を押します。
    - 変更を適用しますが、リストにスタッシュを **残します**。
3.  **ポップ (Pop)**: スタッシュを選択し、`backspace` を押します。
    - 変更を適用し、リストからスタッシュを **削除します**。
4.  **削除 (Drop)**: スタッシュを選択し、`d` を押します。
    - スタッシュを **完全に削除します**。
