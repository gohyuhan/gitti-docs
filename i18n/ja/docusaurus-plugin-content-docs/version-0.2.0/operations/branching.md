# ブランチ管理

**ローカルブランチ**パネルから Git ブランチを直接管理します。

## キーバインディング

| キー    | アクション                 |
| :------ | :------------------------- |
| `n`     | 新しいブランチを作成       |
| `enter` | 選択したブランチに切り替え |
| `d`     | 選択したブランチを削除     |
| `r`     | 選択したブランチ名を変更   |

## ワークフロー

### 新しいブランチの作成

1.  **トリガー**: **ローカルブランチ**パネルで `n` を押します。
2.  **ポップアップ**: **新しいブランチタイプの選択**ポップアップが表示されます。
3.  **モードの選択**:
    - **New Branch**: ブランチを作成しますが、現在のブランチに留まります。
    - **New Branch & Switch**: ブランチを作成し、すぐに切り替えます。
    - **New Branch From Remote**: リモートブランチを選択して、それをベースに新しいブランチを作成します。
4.  **次のポップアップ**:
    - "From Remote" を選択した場合、最初にリモートブランチを選択します。
    - その後、**新しいブランチの作成**入力ポップアップが表示されます。
5.  **入力**: 新しいブランチ名を入力します。
6.  **送信**: `enter` を押します。

---

### 別のブランチへの切り替え

1.  **選択**: **ローカルブランチ**パネルで `↑`/`↓` または `j`/`k` を使用してブランチを選択します。
2.  **トリガー**: `enter` を押します。
3.  **ポップアップ**: コンフリクトや未コミットの変更がある場合、**ブランチ切り替えタイプの選択**ポップアップが表示されます。
4.  **出力**: **ブランチ切り替え出力**ポップアップに結果が表示されます。

---

### ブランチの削除

1.  **選択**: 削除したいブランチをハイライトします。
2.  **トリガー**: `d` を押します。
3.  **確認**: 確認ポップアップが表示されます（強制削除が必要な場合など）。

---

### ブランチ名の変更

1.  **選択**: 名前を変更したいブランチをハイライトします。
2.  **トリガー**: `r` を押します。
3.  **入力**: 新しい名前を入力するポップアップが表示されます。
4.  **送信**: `enter` を押します。
