# 簡介

**Gitti** 是一個快速、輕量級的 Git 操作終端 UI。它專為保持您的工作流而設計，透過鍵盤驅動的介面來執行常見的 Git 任務。

## 核心理念

Gitti 圍繞三個主要組件構建：
1.  **本地分支 (Local Branches)** (左上)
2.  **修改的檔案 (Modified Files)** (左中)
3.  **暫存區 (Stash)** (左下)

**詳細視圖 (Detail View)** (右側面板) 會根據您的選擇上下文進行更新。

## 全局導航

| 鍵 | 動作 |
| :--- | :--- |
| `1` | 切換焦點到**本地分支** |
| `2` | 切換焦點到**修改的檔案** |
| `3` | 切換焦點到**Stash** |
| `tab` | 向前切換組件 |
| `shift+tab` | 向後切換組件 |
| `?` | 打開全局快捷鍵幫助 |

## 快速開始 {#getting-started}

1.  打開您的終端。
2.  導航到一個 Git 倉庫。
3.  運行 `gitti`。

您將看到介面已填充了當前倉庫的狀態。使用數字鍵 (`1`, `2`, `3`) 在面板之間跳轉。
