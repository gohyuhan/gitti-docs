# 暫存 (Stash)

保存未提交的更改以備後用。

## 導航
按 `3` 聚焦 **暫存區** 面板。

## 快捷鍵

| 按鍵 | 動作 | 上下文 |
| :--- | :--- | :--- |
| `s` | 暫存檔案 | 從修改的檔案面板 |
| `S` | 暫存所有 | 從修改的檔案面板 |
| `space` | 應用暫存 (Apply) | 從暫存區面板 |
| `backspace` | 彈出暫存 (Pop) | 從暫存區面板 |
| `d` | 丟棄暫存 (Drop) | 從暫存區面板 |

## 工作流

### 創建暫存

1.  **聚焦**: 導航到 **修改的檔案** (`2`)。
2.  **觸發**:
    - 按 `s` 僅暫存 **選中的檔案**。
    - 按 `S` 暫存 **所有檔案**。
3.  **彈窗**: "Stash Message" (暫存訊息)。輸入描述性名稱。
4.  **提交**: 按 `enter`。

---

### 管理暫存

1.  **聚焦**: 導航到 **暫存區** (`3`)。
2.  **應用 (Apply)**: 選中一個暫存並按 `space`。
    - 應用更改但 **保留** 列表中的暫存。
3.  **彈出 (Pop)**: 選中一個暫存並按 `backspace`。
    - 應用更改並從列表中 **移除** 暫存。
4.  **丟棄 (Drop)**: 選中一個暫存並按 `d`。
    - **永久刪除** 該暫存。
