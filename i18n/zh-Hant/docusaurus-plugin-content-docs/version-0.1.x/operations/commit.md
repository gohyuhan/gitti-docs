# 提交更改

使用專用輸入介面創建和修改提交。

## 快捷鍵

| 按鍵 | 動作 |
| :--- | :--- |
| `c` | 提交 (Commit) |
| `A` | 修改提交 (Amend) |
| `ctrl+e` | 提交表單 |
| `esc` | 取消 |
| `tab` | 下一個欄位 |
| `shift+tab` | 上一個欄位 |

## 工作流

### 創建提交

1.  **暫存**: 確保您已在 **修改的檔案** 面板中暫存了檔案。
2.  **觸發**: 按 `c`。
3.  **彈窗**: **提交彈窗** 出現，包含兩個欄位：
    - **Summary**: 提交資訊的摘要（第一行）。
    - **Description**: 提交資訊的詳細描述。
4.  **輸入**:
    - 輸入摘要。
    - 按 `tab` 移動到描述（可選）。
5.  **提交**: 按 `ctrl+e`。
    - **為什麼是 `ctrl+e`?**: 因為 `enter` 用於在描述欄位中換行，所以使用 `ctrl+e` 來提交表單。

### 修改提交 (Amend)

1.  **觸發**: 按 `A` (shift+a)。
2.  **彈窗**: **修改提交彈窗** 打開，預填了 *上一次* 的提交資訊。
3.  **修改**: 編輯資訊，或者直接提交以包含新暫存的檔案。
4.  **提交**: 按 `ctrl+e`。
