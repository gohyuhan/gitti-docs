# 分支管理

高效管理您的本地分支。Gitti 透過引導式流程處理分支的創建和切換。

## 導航
按 `1` 聚焦 **本地分支** 面板。

## 快捷鍵

| 按鍵 | 動作 | 上下文 |
| :--- | :--- | :--- |
| `enter` | 切換分支 | 選中的分支未檢出時 |
| `n` | 創建分支 | 在分支面板中始終可用 |

## 工作流

### 切換分支

1.  **選擇**: 使用 `up`/`down` (`k`/`j`) 導航到您想要檢出的分支。
2.  **觸發**: 按 `enter`。
3.  **邏輯流程**:
    - **如果分支已檢出**: 無操作。
    - **如果分支未檢出**: 出現彈窗詢問切換方式（例如：標準檢出）。
4.  **確認**: 選擇切換類型並按 `enter`。

### 創建新分支

1.  **觸發**: 按 `n`。
2.  **步驟 1: 選擇類型**: 出現彈窗詢問創建模式：
    - `New Branch`: 僅創建。
    - `New Branch & Switch`: 創建並立即檢出。
3.  **步驟 2: 輸入名稱**: 選擇類型後，出現新彈窗輸入分支名稱。
4.  **提交**: 輸入名稱並按 `enter`。
