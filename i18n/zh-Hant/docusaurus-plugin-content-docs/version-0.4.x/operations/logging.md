# 日誌與調試

Gitti 提供了一個專用的日誌面板，幫助您對問題進行故障排除或實時監控 Git 操作。

## 訪問日誌面板

按 `/` 聚焦 **日誌面板**。

## 快捷鍵

| 按鍵 | 動作     |
| :--- | :------- |
| `e`  | 導出日誌 |

## 功能

### 日誌導出

您可以將當前的會話日誌導出為文件，以便進行外部分析或錯誤報告。

1.  按 `/` 聚焦日誌面板。
2.  按 `e`。
3.  日誌文件將保存到系統的 **Downloads** 文件夾中。

### 配置

您可以使用命令行標誌自定義日誌行為：

- **最大日誌數量 (Max Log Count)**: 設置保留在內存中的最大日誌條目數。

  ```bash
  gitti --max-log-count 1000
  ```

- **顯示 X 條日誌 (Show X Log)**: 設置在面板中顯示的最近日誌數量。
  ```bash
  gitti --show-x-log 50
  ```
