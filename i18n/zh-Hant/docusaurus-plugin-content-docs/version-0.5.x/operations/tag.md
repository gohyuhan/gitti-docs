# 標籤操作

從 **標籤** 面板直接管理 Git 標籤。

## 快捷鍵

| 按鍵     | 操作             |
| :------- | :--------------- |
| `< / >`  | 切換組件面板     |
| `d`      | 刪除標籤         |
| `ctrl+p` | 推送標籤         |
| `f`      | 獲取標籤         |
| `?`      | 快捷鍵和操作說明 |

:::info
本地分支、標籤和遠端面板共享相同的空間。使用 `< / >` 在它們之間切換。要創建新標籤，請在提交日誌面板中使用 `t` 快捷鍵。
:::

## 工作流

### 刪除標籤

1.  **導航**: 導航到標籤組件 (按 `1`) 或切換到標籤組件 (按 `< / >`)。
2.  **選擇**: 使用 `↑/↓` 選擇要刪除的標籤。
3.  **觸發**: 按 `d` 打開刪除標籤選單。
4.  **選擇模式**: 選擇刪除選項:
    - 刪除本地標籤 (Delete local tag)
    - 刪除遠端標籤 (Delete remote tag)
5.  **確認**: 按 `enter` 確認。
6.  **關閉**: 按 `esc` 關閉輸出。

---

### 推送標籤

1.  **導航**: 導航到標籤組件 (按 `1`) 或切換到標籤組件 (按 `< / >`)。
2.  **選擇**: 使用 `↑/↓` 選擇要推送的標籤。
3.  **觸發**: 按 `ctrl+p` 打開推送標籤選單。
4.  **選擇模式**: 選擇推送選項:
    - 推送選中的標籤 (Push selected tag)
    - 推送所有標籤 (Push all tags)
    - 強制推送選中標籤 (Force push selected tag)
    - 強制推送所有標籤 (Force push all tags)
5.  **確認**: 按 `enter` 確認。
6.  **關閉**: 按 `esc` 關閉輸出。

---

### 獲取標籤

1.  **導航**: 導航到標籤組件 (按 `1`) 或切換到標籤組件 (按 `< / >`)。
2.  **觸發**: 按 `f` 打開獲取標籤選單。
3.  **選擇模式**: 選擇獲取選項:
    - 獲取標籤 (Fetch tag(s))
    - 獲取並修剪標籤 (Fetch tag(s) prune) - 刪除遠端已不存在的標籤
    - 獲取並覆蓋標籤 (Fetch tag(s) overwrite) - 使用遠端的不同標籤覆蓋本地
    - 鏡像標籤 (Fetch tag(s) mirror) - 鏡像所有遠端標籤
4.  **確認**: 按 `enter` 確認。
5.  **關閉**: 按 `esc` 關閉輸出。
