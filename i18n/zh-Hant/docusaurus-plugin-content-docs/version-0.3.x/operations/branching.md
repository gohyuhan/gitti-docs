# 分支管理

直接從 **本地分支** 面板管理 Git 分支。

## 快捷鍵

| 按鍵    | 動作           |
| :------ | :------------- |
| `n`     | 創建新分支     |
| `enter` | 切換到選定分支 |
| `d`     | 刪除選定分支   |
| `r`     | 重命名選定分支 |

## 工作流

### 創建新分支

1.  **觸發**: 在 **本地分支** 面板中按 `n`。
2.  **彈窗**: 出現 **選擇新分支類型** 彈窗。
3.  **選擇模式**:
    - **New Branch**: 創建分支但保留在當前分支。
    - **New Branch & Switch**: 創建並立即切換到新分支。
    - **New Branch From Remote**: 選擇一個遠端分支作為新分支的基礎。
4.  **下一個彈窗**:
    - 如果選擇了 "From Remote"，您將首先選擇遠端分支。
    - 然後，出現 **創建新分支** 輸入彈窗。
5.  **輸入**: 輸入新分支名稱。
6.  **提交**: 按 `enter`。

---

### 切換到另一個分支

1.  **選擇**: 在 **本地分支** 面板中使用 `↑`/`↓` 或 `j`/`k` 選擇一個分支。
2.  **觸發**: 按 `enter`。
3.  **彈窗**: 如果存在衝突或未提交的更改，會出現 **選擇切換分支類型** 彈窗。
4.  **輸出**: **切換分支輸出** 彈窗顯示結果。

---

### 刪除分支

1.  **選擇**: 高亮顯示您要刪除的分支。
2.  **觸發**: 按 `d`。
3.  **確認**: 出現確認彈窗（除非隱含/安全強制刪除）。

---

### 重命名分支

1.  **選擇**: 高亮顯示您要重命名的分支。
2.  **觸發**: 按 `r`。
3.  **輸入**: 出現輸入新名稱的彈窗。
4.  **提交**: 按 `enter`。
