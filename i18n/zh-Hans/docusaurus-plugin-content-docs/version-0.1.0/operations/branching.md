# 分支管理

高效管理您的本地分支。Gitti 通过引导式流程处理分支的创建和切换。

## 导航
按 `1` 聚焦 **本地分支** 面板。

## 快捷键

| 按键 | 动作 | 上下文 |
| :--- | :--- | :--- |
| `enter` | 切换分支 | 选中的分支未检出时 |
| `n` | 创建分支 | 在分支面板中始终可用 |

## 工作流

### 切换分支

1.  **选择**: 使用 `up`/`down` (`k`/`j`) 导航到您想要检出的分支。
2.  **触发**: 按 `enter`。
3.  **逻辑流程**:
    - **如果分支已检出**: 无操作。
    - **如果分支未检出**: 出现弹窗询问切换方式（例如：标准检出）。
4.  **确认**: 选择切换类型并按 `enter`。

### 创建新分支

1.  **触发**: 按 `n`。
2.  **步骤 1: 选择类型**: 出现弹窗询问创建模式：
    - `New Branch`: 仅创建。
    - `New Branch & Switch`: 创建并立即检出。
3.  **步骤 2: 输入名称**: 选择类型后，出现新弹窗输入分支名称。
4.  **提交**: 输入名称并按 `enter`。
