# 提交更改

使用专用输入界面创建和修改提交。

## 快捷键

| 按键 | 动作 |
| :--- | :--- |
| `c` | 提交 (Commit) |
| `A` | 修改提交 (Amend) |
| `ctrl+e` | 提交表单 |
| `esc` | 取消 |
| `tab` | 下一个字段 |
| `shift+tab` | 上一个字段 |

## 工作流

### 创建提交

1.  **暂存**: 确保您已在 **修改的文件** 面板中暂存了文件。
2.  **触发**: 按 `c`。
3.  **弹窗**: **提交弹窗** 出现，包含两个字段：
    - **Summary**: 提交信息的摘要（第一行）。
    - **Description**: 提交信息的详细描述。
4.  **输入**:
    - 输入摘要。
    - 按 `tab` 移动到描述（可选）。
5.  **提交**: 按 `ctrl+e`。
    - **为什么是 `ctrl+e`?**: 因为 `enter` 用于在描述字段中换行，所以使用 `ctrl+e` 来提交表单。

---

### 修改提交 (Amend)

1.  **触发**: 按 `A` (shift+a)。
2.  **弹窗**: **修改提交弹窗** 打开，预填了 *上一次* 的提交信息。
3.  **修改**: 编辑信息，或者直接提交以包含新暂存的文件。
4.  **提交**: 按 `ctrl+e`。
