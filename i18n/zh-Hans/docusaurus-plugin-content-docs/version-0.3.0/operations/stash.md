# 暂存 (Stash)

保存未提交的更改以备后用。

## 导航

按 `4` 聚焦 **暂存区** 面板。

## 快捷键

| 按键        | 动作             | 上下文           |
| :---------- | :--------------- | :--------------- |
| `s`         | 暂存文件         | 从修改的文件面板 |
| `S`         | 暂存所有         | 从修改的文件面板 |
| `space`     | 应用暂存 (Apply) | 从暂存区面板     |
| `backspace` | 弹出暂存 (Pop)   | 从暂存区面板     |
| `d`         | 丢弃暂存 (Drop)  | 从暂存区面板     |

## 工作流

### 创建暂存

1.  **聚焦**: 导航到 **修改的文件** (`2`)。
2.  **触发**:
    - 按 `s` 仅暂存 **选中的文件**。
    - 按 `S` 暂存 **所有文件**。
3.  **弹窗**: "Stash Message" (暂存信息)。输入描述性名称。
4.  **提交**: 按 `enter`。

---

### 管理暂存

1.  **聚焦**: 导航到 **暂存区** (`3`)。
2.  **应用 (Apply)**: 选中一个暂存并按 `space`。
    - 应用更改但 **保留** 列表中的暂存。
3.  **弹出 (Pop)**: 选中一个暂存并按 `backspace`。
    - 应用更改并从列表中 **移除** 暂存。
4.  **丢弃 (Drop)**: 选中一个暂存并按 `d`。
    - **永久删除** 该暂存。
