# 日志与调试

Gitti 提供了一个专用的日志面板，帮助您对问题进行故障排除或实时监控 Git 操作。

## 访问日志面板

按 `/` 聚焦 **日志面板**。

## 快捷键

| 按键 | 动作     |
| :--- | :------- |
| `e`  | 导出日志 |

## 功能

### 日志导出

您可以将当前的会话日志导出为文件，以便进行外部分析或错误报告。

1.  按 `/` 聚焦日志面板。
2.  按 `e`。
3.  日志文件将保存到系统的 **Downloads** 文件夹中。

### 配置

您可以使用命令行标志自定义日志行为：

- **最大日志数量 (Max Log Count)**: 设置保留在内存中的最大日志条目数。

  ```bash
  gitti --max-log-count 1000
  ```

- **显示 X 条日志 (Show X Log)**: 设置在面板中显示的最近日志数量。
  ```bash
  gitti --show-x-log 50
  ```
