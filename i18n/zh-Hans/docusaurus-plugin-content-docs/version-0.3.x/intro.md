# 简介

**Gitti** 是一个快速、轻量级的 Git 操作终端 UI。它专为保持您的工作流而设计，通过键盘驱动的界面来执行常见的 Git 任务。

## 核心理念

Gitti 围绕三个主要组件构建：

1.  **本地分支 (Local Branches)** (左上)
2.  **修改的文件 (Modified Files)** (左中)
3.  **提交日志 (Commit Log)** (左下)
4.  **暂存区 (Stash)** (左下)

**详细视图 (Detail View)** (右侧面板) 会根据您的选择上下文进行更新。

## 全局导航

| 键          | 动作                     |
| :---------- | :----------------------- |
| `1`         | 切换焦点到**本地分支**   |
| `2`         | 切换焦点到**修改的文件** |
| `3`         | 切换焦点到**提交日志**   |
| `4`         | 切换焦点到**暂存区**     |
| `tab`       | 向前切换组件             |
| `shift+tab` | 向后切换组件             |
| `?`         | 打开全局快捷键帮助       |

## 快速开始 {#getting-started}

1.  打开您的终端。
2.  导航到一个 Git 仓库。
3.  运行 `gitti`。

您将看到界面已填充了当前仓库的状态。使用数字键 (`1`, `2`, `3`, `4`) 在面板之间跳转。
