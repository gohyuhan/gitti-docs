# 分支管理

直接从 **本地分支** 面板管理 Git 分支。

## 快捷键

| 按键    | 动作           |
| :------ | :------------- |
| `n`     | 创建新分支     |
| `enter` | 切换到选定分支 |
| `d`     | 删除选定分支   |
| `r`     | 重命名选定分支 |

## 工作流

### 创建新分支

1.  **触发**: 在 **本地分支** 面板中按 `n`。
2.  **弹窗**: 出现 **选择新分支类型** 弹窗。
3.  **选择模式**:
    - **New Branch**: 创建分支但保留在当前分支。
    - **New Branch & Switch**: 创建并立即切换到新分支。
    - **New Branch From Remote**: 选择一个远程分支作为新分支的基础。
4.  **下一个弹窗**:
    - 如果选择了 "From Remote"，您将首先选择远程分支。
    - 然后，出现 **创建新分支** 输入弹窗。
5.  **输入**: 输入新分支名称。
6.  **提交**: 按 `enter`。

---

### 切换到另一个分支

1.  **选择**: 在 **本地分支** 面板中使用 `↑`/`↓` 或 `j`/`k` 选择一个分支。
2.  **触发**: 按 `enter`。
3.  **弹窗**: 如果存在冲突或未提交的更改，会出现 **选择切换分支类型** 弹窗。
4.  **输出**: **切换分支输出** 弹窗显示结果。

---

### 删除分支

1.  **选择**: 高亮显示您要删除的分支。
2.  **触发**: 按 `d`。
3.  **确认**: 出现确认弹窗（除非隐含/安全强制删除）。

---

### 重命名分支

1.  **选择**: 高亮显示您要重命名的分支。
2.  **触发**: 按 `r`。
3.  **输入**: 出现输入新名称的弹窗。
4.  **提交**: 按 `enter`。
